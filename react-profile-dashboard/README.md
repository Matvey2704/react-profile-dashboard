# Профиль пользователя (Dashboard)

Фронтенд-реплика страницы профиля из продуктового проекта.

## Описание

Проект воспроизводит типичный личный кабинет пользователя, где можно управлять объявлениями, отслеживать активность и работать с интерфейсом в разных состояниях.

Задача — показать подход к разработке интерфейсов в реальных условиях:
- структурирование UI
- работа с состояниями
- предсказуемое поведение компонентов

## Функциональность

- Список объявлений (Active / Archive)
- Поиск по объявлениям
- Статистика (активные, архив, просмотры)
- Разные UI-состояния (пустой список, фильтрация)
- Адаптивная верстка

## Стек

- React
- TypeScript
- Vite

## Примечание

Это упрощённая реплика функционала из продуктовой разработки.  
Часть логики и UI изменена в связи с NDA, но сохранены основные паттерны и подходы.
